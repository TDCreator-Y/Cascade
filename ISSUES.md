# Cascade Engine — 问题清单

> 按优先级排序，完成后在方框内标记 `x`。

---

## P0 安全问题（必须修复）

- [x] **#1** 全局禁用 TLS 证书验证 — `cli_proxy.rs` 向系统环境变量写入 `NODE_TLS_REJECT_UNAUTHORIZED=0`，影响所有 Node.js 进程
- [x] **#2** 密码明文存储 — ISP 密码以明文写入 `config.json`，UI 无任何提示
- [x] **#3** CSP 安全策略缺失 — `tauri.conf.json` 中 `"csp": null` 完全禁用内容安全策略

---

## P1 功能缺失（核心缺失）

- [x] **#4** `github.com` 误入直连列表 — 大陆用户访问 GitHub 需要走代理，但当前硬编码为直连
- [x] **#5** 缺少实时连接日志面板 — 代理运行期间 UI 完全黑盒，无法查看路由决策与错误
- [x] **#6** 端口 10808 被占用时服务无声失败 — `start_cascade` 在 `tokio::spawn` 外部返回成功，绑定失败仅打印到 stderr
- [x] **#7** 缺少 VPN 端口连通性预检 — 填错 VPN 端口后点击启动无任何前置提示

---

## P2 用户体验

- [x] **#8** 表单缺少输入验证 — IP 格式、端口范围（1–65535）未在前端校验
- [x] **#9** 停止与启动失败状态无区分 — 两种情况均回到 `idle`，用户无法判断是正常停止还是出错
- [x] **#10** 窗口固定大小无法调整 — `tauri.conf.json` 未设置 `resizable: true`
- [x] **#11** 非 Windows 平台代理功能缺失 — macOS/Linux 仅有空桩，系统代理和 CLI 注入均不生效

---

## P3 工程质量

- [x] **#12** 缺少 ESLint 配置 — 项目无 lint 规则，TypeScript 类型错误以外的问题无法被发现
- [x] **#13** 缺少 Rust 后端单元测试 — `cascade_core.rs`（374 行核心逻辑）零测试覆盖
- [x] **#14** 缺少 CI/CD 流水线 — 无自动化构建与测试，代码质量依赖人工检查
